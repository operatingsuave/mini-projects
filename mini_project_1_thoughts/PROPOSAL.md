## My Project Proposal

**What I'm building:**  
I'm building a simple Python app called Thoughts that lets users save short personal thoughts as markdown files.

**Why I chose this:**  
I chose this because I wanted to build something personal and useful instead of a generic calculator or game. It gives me a simple way to save thoughts, moments, or ideas I want to remember.

**Core features:**  
- Add a thought with a title, place, and short note  
- Save each thought as a markdown file inside a thoughts folder  
- Turn the title into a clean filename for the markdown file  
- Format the saved file so the thought is easy to read later  

**What I don't know yet:**  
- The best way to turn a title into a safe filename  
- How to format the markdown file cleanly  
- How much input validation I should add without making the program too complex
