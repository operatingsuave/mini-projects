## My Project Proposal

**What I'm building:**  
I'm building a simple Python app called Thoughts that lets users write short personal thoughts and save them as markdown files.

**Why I chose this:**  
I chose this because I wanted to build something personal and useful instead of a generic calculator or game. It gives me a simple way to save thoughts, moments, or ideas I want to remember.

**Core features:**  
- Show a simple menu with options to add a thought, view saved thoughts, or quit  
- Let the user enter a title, place, and short note  
- Save each thought as a markdown file inside the `thoughts` folder  
- Keep running in a loop so the user can continue using the app without restarting  

**What I don't know yet:**  
- The best way to turn a title into a safe filename  
- How to format the markdown file cleanly  
- How much input validation I should add without making the program too complex
