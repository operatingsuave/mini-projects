# Thoughts

Thoughts is a simple Python app that lets users save short personal thoughts as markdown files.

## What It Does

The program asks the user for:
- a title
- a place
- a short note

It then saves that thought as a `.md` file inside the `thoughts` folder.

## Features

- Takes user input
- Creates a markdown file for each thought
- Uses the thought title to create a filename
- Saves the thought in a clean, readable format

## How to Run

From inside the `mini_project_1_thoughts` folder, run:

```bash
python thoughts.py
